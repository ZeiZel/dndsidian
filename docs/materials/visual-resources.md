# Визуальные ресурсы

## Battlemaps

### Коллекции карт

- **[Lost Atlas](https://lostatlas.co)** — высококачественные карты
- **[r/battlemaps](https://www.reddit.com/r/battlemaps/hot/)** — сообщество с бесплатными картами
- **[Czepeku](https://www.czepeku.com/fantasy/maps)** — детальные карты с вариациями
- **[DnDavid Drive](https://drive.google.com/drive/folders/1QiBxKfHjNdYvmuw6mMfigqunJZrn50QH)** — огромная коллекция
- **[2 Minute Tabletop](https://2minutetabletop.com)** — быстрые карты
- **[Kemono](https://kemono.cr/artists?q=battlemap&service=&sort_by=favorited&order=)** — NSFW, платный контент

### Создание своих карт

- **[Dungeondraft](https://dungeondraft.net/)** — лучший инструмент для карт подземелий
- **[Inkarnate](https://inkarnate.com/)** — world maps и региональные карты
- **[Dungeon Scrawl](https://dungeonscrawl.com/)** — быстрое создание карт
- **[Tom Cartos Patreon](https://www.patreon.com/tomcartos)** — профессиональные карты
- **[Forgotten Adventures](https://www.forgotten-adventures.net/)** — ассеты и карты

### Как готовить карты

**Оптимизация:**
- Конвертируй в WEBP (меньше размер)
- Разрешение: 70-100 DPI для VTT
- Размер клетки: 70-140 пикселей

### Генераторы токенов

- **[Token Stamp](https://rolladvantage.com/tokenstamp/)** — простой генератор

- **[Stable Diffusion](https://stablediffusion.com/)** — AI арты (бесплатно)
