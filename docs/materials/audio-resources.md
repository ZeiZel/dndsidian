# Аудио-ресурсы

## Музыка и эмбиенты

**Основные платформы:**

- **[Tabletopaudio](https://tabletopaudio.com/)** — готовые эмбиенты для D&D
- **[Syrinscape](https://syrinscape.com/)** — профессиональные звуки ($10/месяц)
- **[Incompetech](https://incompetech.com/music/)** — бесплатная музыка
- **[Michael Ghelfi YouTube](https://www.youtube.com/@MichaelGhelfi)** — атмосферные треки
- **[Adrian von Ziegler YouTube](https://www.youtube.com/@AdrianvonZiegler)** — фэнтези музыка
- **[Battlebards](https://battlebards.com/)** — звуковые эффекты

## Категории эмбиентов

**По локации:**

- Таверна
- Лес
- Бой
- Подземелье
- Город
- Мистическая локация

**Поиск на YouTube:** "D&D ambience [тип локации]"
