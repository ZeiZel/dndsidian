# Инструменты и генераторы

## Универсальные генераторы

- **[Donjon](https://donjon.bin.sh/)** — генераторы ВСЕГО (имена, сокровища, подземелья, NPC)
- **[Masterthedungeon](https://masterthedungeon.com/tools/)** — коллекция инструментов

## Специализированные

### Генераторы имён
- **[Fantasy Name Generators](https://www.fantasynamegenerators.com/)** — имена для всех рас и культур

### NPC и персонажи
- **[NPC Generator](https://www.kassoon.com/dnd/npc-generator/)** — случайные NPC
- **[5e Character Sheet](https://www.dndbeyond.com/)** — создание персонажей

### Подземелья и локации
- **[Tavern Generator](https://www.eigengrausgenerator.com/)** — генерация таверн и городов
- **[Dungeon Generator](http://www.gozzys.com/dungeon-generator)** — случайные подземелья

### Баланс боёв
- **[Kobold Fight Club Plus](https://koboldplus.club/)** — расчёт сложности боя

## Трекеры инициативы

- **[Improved Initiative](https://improvedinitiative.app/)** — онлайн трекер
