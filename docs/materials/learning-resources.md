## Справочники онлайн

- **[D&D Beyond](https://www.dndbeyond.com/)** — официальный цифровой контент
- **[Roll20 Compendium](https://roll20.net/compendium/dnd5e)** — бесплатный SRD
- **[5e.tools](https://5e.tools/)** — полный справочник (неофициальный)
- **[DNDsu](https://dndsu.ru/)** — русский справочник
- **[5e14](https://5e14.ru/)** — ещё один русский ресурс
